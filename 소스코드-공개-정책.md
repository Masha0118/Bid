# Source Code Policy

이 저장소는 설계와 문제 해결 과정을 설명하는 공개 포트폴리오입니다.

## 포함하는 내용

- 시스템 구성요소와 책임
- 기술 선택의 배경, 장점과 한계
- 실제 코드와 테스트에서 확인한 구현 범위
- 일반화한 흐름도와 의사 코드
- 시연·스크린샷 촬영 가이드

## 포함하지 않는 내용

- 운영 소스코드, 클래스·메서드 전체, Lua 원문
- 실제 API 경로와 전체 명세
- SQL, DDL, 테이블·인덱스 상세 정의
- 비밀값, 인증 헤더, 내부 주소, 메시지 채널·Redis 키 이름
- 실제 사용자·관리자·인프라 식별 정보

공개 문서의 예시는 구현 원리를 설명하기 위한 일반화된 표현입니다. 원본 소스 접근이 필요한 코드 리뷰는 보안 범위와 방법을 별도로 합의한 경우에만 진행합니다.
