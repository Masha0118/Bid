# 화면 자산

이 폴더에는 현재 Bid On 웹 화면과 일치하는 대표 이미지와 GIF만 둡니다.

## 권장 파일

- `overview.webp`: 경매 체험 전체 화면
- `direct-bid.gif`: 직접 입찰과 결과 갱신
- `concurrent-bidding.gif`: 가상 참가자 동시 입찰
- `participant-sync.gif`: 참가자 카드 상태 동기화
- `processing-flow.gif`: 실제 데이터 처리 과정

현재는 배포 화면의 실행 기능이 비활성 상태이므로 대표 자산을 추가하지 않았습니다. 연결이 활성화된 화면을 촬영한 뒤 README에 실제 존재하는 파일만 연결합니다.

## 추가 전 확인

- 개인정보와 내부 식별자 제거
- 내부 주소, 인증값과 로컬 경로 제거
- 현재 버튼명과 화면 구성이 문서 설명과 일치하는지 확인
- 직접 제작했거나 사용할 권한이 있는 자산만 추가
- GIF의 첫 프레임만 보아도 어떤 기능인지 알 수 있게 구성
