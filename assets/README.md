# Assets

`flow-console-poster.png`는 공개용 가상 데이터만 사용하는 Flow Console의 1440×900 대표 이미지입니다. 실제 Flutter UI, 운영 로그와 사용자 정보는 포함하지 않습니다.

공개용 다이어그램 렌더링 파일, 비식별 썸네일과 영상 표지를 둘 위치입니다. 원본 서비스 이미지나 업로드 파일은 자동 복사하지 않습니다.

추가 전 확인 사항:

- 사용자·관리자·사업자 식별 정보 제거
- 내부 URL, 토큰, 파일 경로 제거
- 직접 제작했거나 공개 권한이 있는 자산만 사용
- 다이어그램 원본과 렌더링 결과가 일치하는지 확인
