# 기능 상태

상태는 코드 존재와 테스트 실행 여부를 구분해 보수적으로 분류했습니다.

| 기능 | 상태 | 근거 요약 |
|---|---|---|
| Redis Lua 입찰 판정 | 구현됐으나 추가 검증 필요 | 서비스 호출과 규칙 테스트 코드 확인, 이번 분석에서 미실행 |
| 자동 연장과 경매 버전 증가 | 구현됐으나 추가 검증 필요 | Lua 상태 변경과 클라이언트 처리 확인 |
| WebSocket 버전·중복·누락 처리 | 구현됐으나 추가 검증 필요 | 서버 버퍼·Flutter 판정·테스트 코드 확인 |
| 종료 잠금과 완료 결과 확인 | 구현됐으나 추가 검증 필요 | Redis 잠금, 결과 요약, 재시도 테스트 코드 확인 |
| 낙찰 거래 중복 방지 | 구현됐으나 추가 검증 필요 | 상품 행 잠금·기존 거래 조회·테스트 코드 확인 |
| 종료 실패 시 상태 유지 | 구현됐으나 추가 검증 필요 | Main API 실패 시 조기 반환과 테스트 코드 확인 |
| RabbitMQ 시작·종료 작업 | 구현됐으나 추가 검증 필요 | 발행·리스너·재시도 구성과 테스트 코드 확인 |
| Kafka 입찰 감사 | 구현됐으나 추가 검증 필요 | ACK, 재시도, 로컬 실패 보관, 중복 소비와 통합 테스트 코드 확인 |
| JWT·내부 요청 인증 | 구현됐으나 추가 검증 필요 | 서버별 보안 필터와 Stateless 설정 확인 |
| 요청 횟수 제한 | 구현됐으나 추가 검증 필요 | Redis·메모리 카운터와 단위 테스트 코드 확인 |
| 관리자 광고·신고·분쟁·회원 제재 | 구현됐으나 추가 검증 필요 | 컨트롤러·서비스·관리 화면 확인, 전체 흐름 미실행 |
| Batch 지표·위험 분석 | 구현됐으나 추가 검증 필요 | Job·스케줄러·계산·저장 코드와 다수 테스트 확인 |
| AI 상품 등록 | 부분 구현 | 코드와 테스트가 있으나 현재 작업 트리에 진행 중 변경 존재 |
| AI 고객센터 | 부분 구현 | 검색·답변·관리자 검토 코드 존재, 품질 평가 결과 미확인 |
