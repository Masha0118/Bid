# 기능 상태

상태는 코드 존재와 테스트 실행 여부를 구분해 보수적으로 분류했습니다.

| 기능 | 상태 | 근거 요약 |
|---|---|---|
| Redis Lua 입찰 판정 | 코드에서 구현 확인 · 추가 검증 필요 | 서비스 호출과 테스트 코드 확인, 이번 분석에서 미실행 |
| 자동 연장과 경매 버전 증가 | 코드에서 구현 확인 · 추가 검증 필요 | Lua 상태 변경과 클라이언트 처리 확인 |
| WebSocket 버전·중복·누락 처리 | 코드에서 구현 확인 · 추가 검증 필요 | 서버 버퍼·Flutter 판정·테스트 코드 확인 |
| 종료 잠금과 완료 결과 확인 | 코드에서 구현 확인 · 추가 검증 필요 | Redis 잠금, 결과 요약, 재시도 테스트 코드 확인 |
| 낙찰 거래 중복 방지 | 코드에서 구현 확인 · 추가 검증 필요 | 상품 행 잠금·기존 거래 조회·테스트 코드 확인 |
| 종료 실패 시 상태 유지 | 코드에서 구현 확인 · 추가 검증 필요 | Main API 실패 시 조기 반환과 테스트 코드 확인 |
| RabbitMQ 시작·종료 작업 | 코드에서 구현 확인 · 추가 검증 필요 | 발행·리스너·재시도 구성과 테스트 코드 확인 |
| Kafka 입찰 감사 | 코드에서 구현 확인 · 추가 검증 필요 | ACK, 재시도, 로컬 실패 보관, 중복 소비와 통합 테스트 코드 확인 |
| 세션·내부 요청 인증 | 코드에서 구현 확인 · 추가 검증 필요 | 사용자 세션과 서버 간 인증 흐름 확인 |
| 요청 횟수 제한 | 코드에서 구현 확인 · 추가 검증 필요 | Redis·메모리 카운터와 단위 테스트 코드 확인 |
| 관리자 광고·신고·분쟁·회원 제재 | 코드에서 구현 확인 · 추가 검증 필요 | 컨트롤러·서비스·관리 화면 확인, 전체 흐름 미실행 |
| Batch 지표·위험 분석 | 코드에서 구현 확인 · 추가 검증 필요 | Job·스케줄러·계산·저장 코드와 다수 테스트 확인 |
| Flow Console | 흐름 재구성 · 테스트 실행 확인 | 딥링크 테스트와 정적 빌드 실행 |
| AI 상품 등록 | 부분 구현 | 코드와 테스트가 있으나 현재 작업 트리에 진행 중 변경 존재 |
| AI 고객센터 | 부분 구현 | 검색·답변·관리자 검토 코드 존재, 품질 평가 결과 미확인 |

원본 서비스 테스트는 이번 분석에서 실행하지 않았습니다. Flow Console 테스트 실행은 공개용 시각화 도구에만 해당하며 원본 서비스 검증으로 확대해 해석하지 않습니다.
