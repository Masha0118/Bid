# Verified Runs

재현 가능한 실행 결과가 준비되면 이 폴더에 환경 정보와 함께 저장합니다. 현재 실제 결과 파일은 없으며 예제 수치를 실제 결과처럼 작성하지 않습니다.

## 결과 파일에 포함할 항목

- 실행 시각
- 테스트 환경과 배포 사양
- 부하 발생기와 서버의 위치
- 참가자 수와 입찰 속도
- 테스트 시간과 전체 요청 수
- 반영된 입찰과 조건상 미반영
- 시스템 오류와 응답 시간 초과
- 평균 응답 시간, p95와 처리량
- 최종 최고가와 최고 입찰자 일치 여부
- 참가자 상태 Version 일치 여부
- Kafka 전달 요청과 저장 완료 중 확인한 범위

## 해석 원칙

- 성능 수치는 실행 환경과 설정을 함께 봅니다.
- 조건상 미반영과 시스템 오류를 분리합니다.
- 확인하지 못한 값은 0이 아니라 `확인 정보 없음`으로 기록합니다.
- 단일 실행 결과를 장시간 안정성이나 최대 처리 한계로 일반화하지 않습니다.
- 같은 설정을 반복 실행해 편차와 재현성을 확인합니다.
