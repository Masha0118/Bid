# 시연 영상 촬영 가이드

실제 영상은 아직 포함하지 않습니다. 촬영 후 개인정보와 내부 주소를 가리고 아래 위치에 링크를 추가합니다.

| 영상 | 권장 길이 | 대본 |
|---|---:|---|
| 전체 사용자 흐름 | 3~5분 | [전체 서비스 시연 대본](01-전체-서비스-시연-대본.md) |
| 동시 입찰 | 약 1분 | [동시 입찰 시연 대본](02-동시-입찰-시연-대본.md) |
| WebSocket 복구 | 1~2분 | [실시간 복구 시연 대본](03-실시간-복구-시연-대본.md) |
| 경매 종료 | 1~2분 | [경매 종료 시연 대본](04-경매-종료-시연-대본.md) |
| 자동화 테스트 | 1~2분 | [테스트 시연 대본](05-테스트-시연-대본.md) |

촬영 전에는 테스트 계정, 테스트 상품과 비식별 주소만 사용하고 브라우저 주소창·터미널 환경변수·토큰·알림 개인정보가 보이지 않는지 확인합니다.
